Guia Instalação da Aplicação

1. Execute o arquivo "Installer.cmd". Após este processo mesmo não sendo necessário, recomendamos que reiniciei a máquina caso não apareça o icone da App.

2. Caso já exista um diretório "PhotoOrganizer" com os arquivos no "C:\Program Files\" e queria instalar novamente... 
basta apertar "Y" sem as aspas no Prompt para sobescrever os arquivos mais antigos, "N" para não sobrescrever e "T" para Todos.

3. Após este processo, uma caixa de mensagem informando o fim da instalação irá surgir na tela, clique em "OK" ou no "X" para fechar a tela com a mensagem.
